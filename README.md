# TEST_OCR

## FE 개발 환경

### Setting

```sh
# Need to use node.js version 18.x.x
# @see https://github.com/nvm-sh/nvm
nvm use 18.x.x
npm install
```

### express app 실행 (with nodemon)

- test tesseract : `npm run test:tesseract_ocr-nlp`
- test easy ocr : `npm run test:easy_ocr`
- dev : `npm run dev`
